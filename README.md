# Customer Support Chatbot

## Overview
This project is a Customer Support Chatbot built using Python and Streamlit.

## Features
- Order Tracking
- Order Cancellation
- Shipping Information
- Refund Policy
- Return Policy
- Damaged Product Complaint Registration
- Payment Methods Information
- Contact Support Information
- Business Hours Information

## Technologies Used
- Python
- Streamlit

## Installation

pip install -r requirements.txt

python -m streamlit run app.py
